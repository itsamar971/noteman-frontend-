# NoteMan — Your Academic Resource Repository

NoteMan is a premium, brutalist-inspired academic platform designed to streamline access to study materials, notes, textbooks, and previous year's exam papers. Built with a focus on speed, aesthetics, and community, NoteMan serves as the ultimate study companion for students and educators.

![NoteMan Banner](/public/navbar-logo.svg)

## 🚀 Key Features

- **Brutalist Dark Mode**: A custom, high-contrast design system optimized for late-night study sessions.
- **Academic Structure**: Intuitively organized by Branch → Semester → Subject.
- **Instant Search**: Find any resource across the repository in milliseconds.
- **Admin Dashboard**: Secure, session-based management for uploading and managing academic resources.
- **Community Focused**: Integrated links to join academic discussion groups.

## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Framer Motion, Lucide Icons.
- **Backend**: Node.js, Express, Passport.js (local session auth), Drizzle ORM, PostgreSQL.
- **Infrastructure**: Supabase (Database & Storage), Vercel (Frontend Hosting), Render (Backend Hosting).

## 📦 Project Structure

```text
frontend/
├── src/
│   ├── components/  # Atomic UI components and features
│   ├── hooks/       # Custom React hooks (Auth, DarkMode, etc.)
│   ├── pages/       # Page-level components
│   └── lib/         # Utility functions and API clients

```
